# Student Performance Analysis using Pandas and Matplotlib

This project was completed as part of my Machine Learning Bootcamp. The objective of this assignment was to practice the fundamentals of Python data analysis using the Pandas library and data visualization using Matplotlib.

In this project, I:

* Created and managed a dataset of student records.
* Performed basic data exploration and analysis using Pandas.
* Added new features such as total scores and eligibility status.
* Filtered data based on specific conditions.
* Identified high-performing students using logical operations.
* Generated various visualizations, including line charts, bar charts, and pie charts using Matplotlib.

### Technologies Used

* Python
* Pandas
* Matplotlib

### Learning Outcomes

Through this project, I gained hands-on experience with:

* DataFrame creation and manipulation
* Feature engineering
* Data filtering and selection
* Basic data visualization techniques
* Exploratory Data Analysis (EDA)

This project serves as an introduction to data analysis and visualization workflows commonly used in Machine Learning and Data Science.
